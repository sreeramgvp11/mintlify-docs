---
title: "Delete Connector Resource"
description: "Remove a resource from a connector and stop syncing it."
openapi: "api-reference/v2/openapi.json DELETE /connectors/{id}/resources/{resource_id}"
---

Removes a configured resource and stops syncing it. Objects already ingested into the knowledge store are not removed.

<RequestExample>

```bash cURL
curl -X DELETE 'https://api.hydradb.com/connectors/{id}/resources/{resource_id}' \
  -H "Authorization: Bearer $HYDRA_DB_API_KEY" \
  -H "API-Version: 2"
```

</RequestExample>

## Path parameters

| Name | Description |
| --- | --- |
| `id` | Connector UUID returned by `POST /connectors`. |
| `resource_id` | Resource identifier (from `GET /connectors/:id/resources`). |

<ResponseExample>

```json 200
{
  "connector_id": "{connector_id}",
  "deleted": true,
  "resource_id": "{resource_id}"
}
```

</ResponseExample>
