---
title: "List Connector Resources"
description: "List configured resources and their sync state."
openapi: "api-reference/v2/openapi.json GET /connectors/{id}/resources"
---

import { Field } from "/snippets/field.jsx";

Returns all resources configured on a connector and their current sync state. Poll `provider_cursor` after triggering a sync  -  a non-empty value confirms the sync ran.

<RequestExample>

```bash cURL
curl 'https://api.hydradb.com/connectors/{id}/resources' \
  -H "Authorization: Bearer $HYDRA_DB_API_KEY" \
  -H "API-Version: 2"
```

</RequestExample>

## Path parameters

| Name | Description |
| --- | --- |
| `id` | Connector UUID returned by `POST /connectors`. |

<ResponseExample>

```json 200
{
  "resources": [
    {
      "resource_id": "C04AKFHN78S",
      "resource_type": "channel",
      "name": "general",
      "sub_tenant_id": "all-hands",
      "provider_cursor": "1717200000.000100",
      "sync_status": "active",
      "last_synced_at": "2026-06-01T12:05:00Z"
    }
  ]
}
```

</ResponseExample>

Use `sync_status` and `error_message` to detect sync failures.
