---
title: "List Sub-Tenants"
description: "List sub-tenant IDs inside a tenant."
openapi: "api-reference/v2/openapi.json GET /tenants/sub-tenants"
---

1. A default sub-tenant is provisioned automatically when a tenant is created. This default sub tenant stores all your context. Create additional sub-tenants at any time to scope data to users, teams, or projects.
2. **Implicit creation.** Sub-tenants are auto-created when ingestion writes data under a new `sub_tenant_id`. The returned list grows organically as your application writes data under new values.

<RequestExample>

```python Python SDK
response = client.tenants.sub_tenants(tenant_id="your tenant id")
```

```typescript TypeScript SDK
const response = await client.tenants.subTenants({
  tenantId: "my_first_tenant",
});
```

```bash cURL
curl -X GET 'https://api.hydradb.com/tenants/sub-tenants?tenant_id=my_first_tenant' \
  -H "Authorization: Bearer <your_api_key>" \
  -H "API-Version: 2"
```

</RequestExample>
